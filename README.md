# Tailwind Starter Project

This is a **very simple starter template** for experimenting with Tailwind CSS and learning to use Git and GitHub.

## What's included

- **`index.html`**: A single page with:
  - A basic header
  - A hero section with a stock photo
  - A "Hello world" style intro line
  - A short description of what the project is for
- **Tailwind via CDN**: No build tools or bundlers — you can just open `index.html` in a browser.

## How to run it

1. Open the project folder.
2. Double‑click `index.html` (or open it in your browser).

You should see the header, image, and intro text styled with Tailwind.

## Getting ready for Git & GitHub

You can initialize a git repository from this folder:

```bash
git init
git add .
git commit -m "Initial Tailwind starter template"
```

Then create a new repository on GitHub and follow their instructions to connect your local repo (usually `git remote add origin ...` and `git push -u origin main`).

As you build your project over time, you can:

- Edit `index.html` to change the content.
- Split things into more HTML files or move to a framework (like React + Vite) later if you want.
- Commit and push changes regularly to track your progress.
